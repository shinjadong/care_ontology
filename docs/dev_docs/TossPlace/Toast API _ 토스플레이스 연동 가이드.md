## Toast API

Toss POS에서 사용자에게 알림이나 안내 메시지를 표시하기 위한 API입니다. 플러그인 상태 변경, 오류 발생, 작업 완료 등의 상황에서 사용자에게 즉각적인 피드백을 제공할 수 있습니다.

## Methods

### open

토스트 메시지를 화면에 표시합니다.
